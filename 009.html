กรรม (ต่อ)


ถ้ากินปลา "กินปลา 3 ตัว" นั้นหมายความว่า คำว่า "ปลา" มีจำนวนมากกว่า 1 เป็นพหูพจน์จะต้องใช้ตัวลงท้าย -j
โดยจะเติมหลังท้ายคำนาม -o แล้วปิดท้ายด้วย -n
-o+j+n
-os -as -is
Mi manĝas 3 fiŝojn. - ฉันกำลังกินปลา 3 ตัว
Vi manĝas 3 fiŝojn. - คุณ/พวกคุณกำลังกินปลา 3 ตัว
Ni manĝas 3 fiŝojn. - พวกเรากำลังกินปลา 3 ตัว
Li manĝas 3 fiŝojn. - เขากำลังกินปลา 3 ตัว
Ŝi manĝas 3 fiŝojn. - เธอกำลังกินปลา 3 ตัว
Ili manĝas 3 fiŝojn. - พวกเขากำลังกินปลา 3 ตัว
Kato manĝas 3 fiŝojn. - แมว 1 ตัวกำลังกินปลา 3 ตัว
Hundoj manĝas 3 fiŝojn. - สุนัขหลายตัวกำลังกินปลา 3 ตัว
